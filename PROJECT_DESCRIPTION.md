# Project Description – Linux Patch Audit

## Goal
The goal of this project is to automate patch auditing on Linux systems to identify pending updates and highlight security-relevant packages.

## Problem
Unpatched systems are one of the main causes of security breaches. System administrators and SRE teams must regularly check for updates and security patches.

## Solution
This tool automates:
- Checking for pending updates
- Identifying security-related packages
- Generating audit reports for documentation

## Skills Demonstrated
- Python scripting
- Bash scripting
- Linux system administration
- Automation
- Security patch management
- Report generation
- Working with system commands

## Real-World Relevance
This type of tool is relevant for:
- Site Reliability Engineering (SRE)
- Security Operations Center (SOC)
- System Administration
- Vulnerability Management
